视频特效助手 - Windows版  v1.0.0

====================================
  首次使用（3步搞定）
====================================

前置条件：安装 Node.js 18+（https://nodejs.org）

1. 双击 setup.bat     → 安装依赖+下载ffmpeg
2. 双击 start.bat     → 启动应用
3. 或命令行运行：npm start

====================================
  两种模式
====================================

模式A - 口播加特效：人在右边，关键词特效在左边，深色遮罩跟随
模式B - 配字幕特效：字幕在底部，关键词特效在左上角同步显示

====================================
  使用流程
====================================

1. 选择模式（口播加特效 / 配字幕特效）
2. 导入视频（拖拽或点击选择）
3. 粘贴口播文案 → AI分析生成特效方案
4. 预览确认（低画质快速预览）
5. 导出视频（可选画质/帧率）

====================================
  打包成exe
====================================

双击 一键打包.bat，安装包在 dist\ 目录

====================================
  AI服务（5种引擎可选）
====================================

- DeepSeek（推荐，便宜）：platform.deepseek.com
- 通义千问：dashscope.console.aliyun.com
- OpenAI：platform.openai.com
- Anthropic Claude：console.anthropic.com
- 本地Ollama：需先安装Ollama，支持qwen3等模型

====================================
  语音转文字（3种方案）
====================================

1. 手动粘贴口播文案（最快最准确）
2. 安装 whisper.cpp 到 bin\ 目录实现本地转录
3. 使用剪映/PR等工具导出字幕文本后粘贴

====================================
  常见问题
====================================

Q: bat窗口一闪就没了？
A: 右键"以管理员身份运行"，或打开cmd手动执行

Q: 提示"ffmpeg未找到"？
A: 首次运行会自动提示下载，或双击 download-ffmpeg.bat

Q: 下载ffmpeg很慢？
A: 需要联网下载约80MB，建议开代理或耐心等待

Q: npm install 失败？
A: 设置国内镜像后重试：
   npm config set registry https://registry.npmmirror.com

Q: 打包后运行报错？
A: 确保打包前已下载ffmpeg（一键打包.bat会自动处理）

Q: 预览时动画不显示？
A: GSAP已内置离线可用，确保网络不影响渲染
